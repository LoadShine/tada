# Tada - 现代、离线优先的智能任务管理应用

[Read in English](README.md)

**Tada** 不仅仅是一个待办事项列表。它是一个为您精心打造的、完全在您浏览器中运行的智能、私密且可高度定制的任务管理空间。结合了强大的 AI 助手和流畅的用户体验，帮助您以前所未有的方式规划和总结您的工作。

## ✨ 核心功能

*   **🚀 离线优先，尊重隐私**:
    *   **无需注册，打开即用**。所有数据都安全地存储在您的本地浏览器中。
    *   您的任务、列表和设置完全由您掌控，绝不上传到任何服务器。

*   **🤖 强大的 AI 助手**:
    *   **AI 创建任务**: 只需用自然语言描述一个任务（例如，“下个月为团队策划一次团建活动”），AI 就能自动为您拆解成包含详细描述、子任务、标签和优先级的结构化任务。
    *   **AI 工作总结**: 根据您选定的已完成和未来任务，一键生成专业、富有洞察力的 Markdown 工作报告。
    *   **高度可配置**: 自由选择并配置您偏好的 AI 服务商，包括 OpenAI、Claude、Gemini、Ollama（本地模型），以及月之暗面、智谱AI、通义千问等多种国内模型。

*   **✍️ 先进的 Markdown 编辑器 (Moondown)**:
    *   内置强大的所见即所得（WYSIWYG）Markdown 编辑器。
    *   支持**气泡菜单**快速格式化，**斜杠命令 (`/`)** 快速插入表格、列表等元素。
    *   集成的**AI续写**功能，助您轻松完成笔记和文档。

*   **✅ 全面的任务组织**:
    *   通过**列表**、**标签**、**截止日期**和**优先级**清晰地组织您的任务。
    *   支持创建**子任务**，将复杂工作分解为简单步骤。
    *   直观的**拖拽排序**功能，轻松调整任务顺序。

*   **📅 多种视图**:
    *   除了经典的列表视图，还提供**日历视图**，让您对截止日期一目了然。

*   **🎨 高度可定制**:
    *   支持**浅色**、**深色**和跟随系统三种外观模式。
    *   内置多种**主题色**，让应用界面符合您的个人风格。
    *   支持**中英文**双语切换。

## 🛠️ 技术栈

*   **前端框架**: React + TypeScript
*   **构建工具**: Vite
*   **状态管理**: Jotai
*   **路由**: React Router
*   **样式**: Tailwind CSS
*   **UI 组件**: Radix UI (Headless) & Lucide Icons
*   **Markdown 编辑器**: Moondown (基于 CodeMirror 6)
*   **拖拽**: @dnd-kit

## 🚀 本地开发

1.  **克隆仓库**:
    ```bash
    git clone https://github.com/your-username/tada.git
    cd tada
    ```

2.  **安装依赖**:
    ```bash
    npm install
    ```

3.  **启动开发服务器**:
    ```bash
    npm run dev
    ```
    应用将在 `http://localhost:5173` (或另一个可用端口) 上运行。

4.  **配置 AI 功能 (可选)**:
    *   应用启动后，点击左下角的设置图标。
    *   进入 "AI 设置" 标签页。
    *   选择您想使用的 AI 服务商，并填入您的 API Key 和其他必要信息。
    *   点击 "测试连接" 以确保配置正确。
